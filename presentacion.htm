Esta es la página de presentación de nuestro proyecto para la VI Olimpiada Informática de la Región de Murcia: un juego de navegador, creado íntegramente desde cero, ambientado en la época romana.

¿En qué consiste?

Un juego de navegador permite a muchos jugadores interactuar en tiempo real accediendo a la web. El objetivo es hacer crecer tu imperio estratégicamente mediante el comercio, la conquista o la diplomacia.

Programación

El proyecto ha sido realizado con PHP como lenguaje de servidor y MySQL como gestor de bases de datos. La programación es orientada a objetos y se sirve de modernas metodologías y patrones. Contiene ¡más de 10000 líneas de código!

Diseño

Para el diseño web se han empleado las últimas tecnologías de HTML5 y CSS3, así como Javascript, mediante su framework Jquery, con el objetivo de crear interfaces atractivas e intuitivas.

Libre y abierto

El código fuente del proyecto se publicará bajo una licencia libre en GitHub tras la presentación. Creemos que cualquiera debe poder utilizarlo: al fin y al cabo, hemos utilizado tecnologías libres.

Es el momento de la práctica. Puedes ver el proyecto accediendo a la página: basta con registrarse con datos de prueba (no hace falta validar el correo) y comenzar a jugar. También puedes descargar el código fuente junto con las bases de datos.
